History of the Website

The Gladys Lacia Instructional Website was created in Google Sites on July 2020 to provide online instructions and downloadable resources for her students during the COVID-19 pandemic. Three years later, on April 2023, this website is being upgraded using HTML, CSS and Javascript programming languages. This website has been operated semi-annually by Ms. Lacia herself since September 2020.
